⚔️ 最佳评论 PK 系统

创新的内容竞争机制，让最有价值的声音被听见

📍
帖子点赞数
156
⚔️
点赞数更高者获胜
💬
最高赞评论
203

🏆 PK 规则

1. 帖子发布用户发表帖子
2. 评论互动用户发表评论
3. 分别点赞帖子和评论各自累赞
4. PK 比较评论点赞数 vs 帖子点赞数
5. 实时更新随点赞数变化实时更新

🎯 创新价值

传统平台中，神评论往往被埋没在海量内容里。TapSpot 的 PK 机制让优质评论能够与帖子本身同台竞技，确保最有价值的声音不会被错过。

公平竞争 实时更新 激励创作

🔌 PK 结果 API

GET/api/posts/:id/best-comment

返回当前 PK 结果：帖子点赞数、最高赞评论信息、PK 胜者

🤖 AI 智能助手

阿里云百炼 Qwen 驱动，地点分析与智能对话

📍 地点智能分析

基于阿里云百炼 Qwen-Turbo 模型，自动分析地点特色和游玩建议

POST/api/ai/analyze

输入：坐标 (lat, lng) + 地点名

输出：地点特色描述 + 游玩建议 + 推荐指数

💬 阿尼亚聊天

与《间谍过家家》阿尼亚角色互动，支持专业问题解答

POST/api/chat

角色：阿尼亚 (Anya)

能力：闲聊 · 问答 · 翻译 · 编程辅助

🗺️
地点分析
坐标+名称 → AI 解读
📝
文字解析
选中文字 → AI 分析
🎯
智能推荐
位置+偏好 → 热门推荐

🔧 AI 服务架构

POST /api/ai/analyze → Backend → 阿里云百炼 API → Qwen-Turbo → 地点描述+建议

未配置 API Key 时，AI 功能返回模拟数据
