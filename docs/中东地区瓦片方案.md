# TapSpot 中东地区地图瓦片方案

## 📋 方案概述

为 TapSpot 平台添加中东地区（伊朗、伊拉克、沙特阿拉伯等）的地图瓦片支持，使用户可以在这些地区正常打卡和查看地图。

## 🗺️ 覆盖区域

### 中东地区范围
- **经度**: 35°E - 65°E
- **纬度**: 12°N - 42°N
- **覆盖国家**:
  - 伊朗 (主要)
  - 伊拉克
  - 沙特阿拉伯
  - 阿联酋
  - 土耳其东部
  - 也门
  - 科威特
  - 巴林
  - 卡塔尔
  - 阿曼

### 缩放级别
- **级别**: 1-10 级
- **用途**:
  - 1-4 级：全球/大洲视图
  - 5-7 级：国家/省份视图
  - 8-10 级：城市视图

## 🔧 技术实现

### 1. 瓦片下载脚本
**文件**: `download_tiles_middle_east.py`

```bash
# 执行下载
python3 download_tiles_middle_east.py
```

**特性**:
- 并发下载 (20 个线程)
- 自动重试机制
- 进度显示
- 断点续传（已存在的瓦片跳过）

### 2. 前端地图配置更新
**文件**: `frontend/src/App.jsx`

**修改内容**:
```jsx
<MapContainer 
  center={[35.8617, 104.1954]} 
  zoom={mapZoom} 
  worldCopyJump={true}  // 新增：支持地图循环滚动
>
  <TileLayer 
    url="https://webrd0{s}.is.autonavi.com/appmaptile?..."
    minZoom={2}          // 修改：支持更低缩放级别
    noWrap={false}       // 新增：允许瓦片跨边界重复
    ...
  />
</MapContainer>
```

### 3. 瓦片存储结构
```
/root/.openclaw/workspace/TapSpot/tiles/
├── 1/          # 缩放级别 1
│   ├── 0/
│   │   ├── 0.png
│   │   └── 1.png
│   └── 1/
├── 2/          # 缩放级别 2
├── ...
└── 10/         # 缩放级别 10
```

## 🚀 使用方式

### 用户端
1. 打开 TapSpot 地图
2. 缩放或拖动到中东地区
3. 正常进行打卡操作
4. 地图会自动加载本地瓦片或在线瓦片

### 开发者
```bash
# 1. 下载瓦片
cd /root/.openclaw/workspace/TapSpot
python3 download_tiles_middle_east.py

# 2. 验证瓦片
ls -la tiles/

# 3. 重启前端服务
cd frontend
npm run dev
```

## ⚠️ 注意事项

1. **瓦片源**: 使用高德地图瓦片源，需要网络可达
2. **存储空间**: 完整瓦片约需 500MB-1GB 存储空间
3. **更新频率**: 建议每月更新一次瓦片数据
4. **法律合规**: 确保地图瓦片使用符合当地法律法规

## 🔍 故障排查

### 问题：地图显示空白
- 检查瓦片目录是否存在：`ls tiles/`
- 检查瓦片文件是否完整：`ls tiles/5/100/`
- 检查浏览器控制台是否有 CORS 错误

### 问题：瓦片下载失败
- 检查网络连接
- 检查高德瓦片源是否可达
- 增加重试次数或降低并发数
